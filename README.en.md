# PyQt5画板

#### Description
PyQt5画板：
（1）支持自由绘画画笔、箭头、矩形、圆形、文本类型
（2）矩形、圆形、箭头画完后均可利用控制点再次缩放/旋转编辑，文本类可双击修改文本，包括字体、大小、粗细、下划线等
（3）支持撤回与重做功能
（4）支持图形轮廓和填充的格式修改
（5）采取双缓冲绘图
（6）支持图元的剪切、复制、粘贴、排列等
（7）支持图片拖拽至画板中
（8）支持画板内容保存为.pgd格式，pgd文件可再次编辑
